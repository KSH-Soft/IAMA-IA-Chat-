<p align="center">IAMA is a lightweight Windows application that takes the form of a small chatbot located at the bottom right of the screen. This chatbot is powered by local AI, is fully configurable, and is designed to be used in both standalone and server modes, enabling large-scale centralized management.</p>

<h2 id="layout">âœ¨ Presentation</h2>

<p align="center">
IAMA is not a general-purpose AI like ChatGPT. It is a local artificial intelligence, controlled and restricted as needed. It is designed to assist users with simple tasks, with a high level of security and control over its responses.
<br>
The main objective of this project is to offer an intelligent, customizable assistance system that can be used in professional, educational, or social environments, while ensuring compliance with an institution's internal rules.
